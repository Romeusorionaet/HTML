# 📘 Blog de Estudo de Tags HTML

Este projeto é uma página simples criada com HTML puro com o objetivo de **praticar e documentar o aprendizado das principais tags HTML**. Cada seção aborda um grupo de elementos com exemplos e descrições.

---

## 🧱 O que você vai encontrar?

🔹 Títulos (`h1` a `h6`)  
🔹 Parágrafos e formatação de texto (`p`, `strong`, `mark`, etc.)  
🔹 Listas ordenadas e não ordenadas (`ol`, `ul`, `li`)  
🔹 Links externos e internos (`a`)  
🔹 Separadores visuais (`hr`)  
🔹 Subíndices e expoentes (`sub`, `sup`)  
🔹 Citações (`blockquote`)  
🔹 Estrutura semântica (`main`, `section`, `article`)  
🔹 Mídias (`img`, `audio`, `video`)  
🔹 Tags adicionais (`abbr`, `font`, `del`)

---

## 🎯 Objetivo

Explorar e entender o funcionamento das principais tags HTML de forma prática e sem depender de frameworks ou CSS avançado.
