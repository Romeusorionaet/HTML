Títulos com <h1> até <h6>

As tags de título, que vão de <h1> até <h6>, são utilizadas para definir a hierarquia de informações em uma página HTML. O <h1> representa o título mais importante (geralmente o título principal da página), enquanto o <h6> é o menos importante, usado para subtítulos ou observações de menor destaque. Essa hierarquia ajuda na organização do conteúdo e também melhora a acessibilidade e o SEO (otimização para mecanismos de busca).

Textos e Parágrafos

As tags de texto em HTML permitem estruturar, destacar e complementar informações dentro de um parágrafo. A tag <p> é usada para criar blocos de texto, enquanto outras tags como <mark>, <small>, <i>, <u> e <strong> são usadas para dar ênfase, alterar o estilo ou destacar partes específicas. Também é possível mostrar texto riscado com <del>, aplicar cores com <font> (mesmo sendo obsoleta), e explicar siglas com <abbr>. Tudo isso contribui para uma melhor apresentação do conteúdo e maior acessibilidade.

Listas Ordenadas e Não Ordenadas

As listas em HTML são usadas para organizar informações de forma estruturada. A tag <ol> cria uma lista ordenada, ou seja, com itens numerados automaticamente, ideal para representar sequências ou passos. Já a tag <ul> cria uma lista não ordenada, onde os itens aparecem com marcadores simples, sendo útil para listas de tópicos ou enumerações livres. Os itens de ambas as listas são definidos com a tag <li>, que representa cada elemento individual da lista.

Lista Ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada:

  • Item A
  • Item B
  • Item C

Links com <a>

A tag <a>, conhecida como "anchor" (âncora), é utilizada para criar hiperlinks em HTML. Com ela, é possível direcionar o usuário para outras páginas, sites, seções dentro da própria página ou até iniciar downloads. O atributo href define o destino do link, e o atributo target="_blank" faz com que o link seja aberto em uma nova aba, melhorando a navegação sem sair da página atual.

Você pode acessar o MDN Web Docs para mais informações sobre HTML.


Separando conteúdo com <hr>

A tag <hr> é usada para criar uma linha horizontal na página, servindo como uma separação visual entre blocos de conteúdo. Ela é útil para indicar mudanças de assunto, transições entre seções ou apenas para melhorar a estética e organização da página. É uma tag autossuficiente, ou seja, não precisa de uma tag de fechamento.

Textos com <sub> e <sup>

As tags <sub> e <sup> são usadas para formatar textos como subíndices e expoentes, respectivamente. A tag <sub> posiciona o texto ligeiramente abaixo da linha base, sendo útil em fórmulas químicas como H2O. Já a tag <sup> posiciona o texto acima da linha base, ideal para indicar potências matemáticas como 102. Essas formatações são especialmente úteis em contextos científicos e acadêmicos.

Citações com <blockquote>

A tag <blockquote> é utilizada para representar citações em bloco, geralmente de outras fontes. É ideal para destacar trechos longos de textos que foram retirados de livros, sites ou autores específicos. O atributo cite pode ser usado para indicar a fonte da citação, tornando o conteúdo mais confiável e organizado.

"O elemento blockquote representa uma seção que é citada de outra fonte."

Estrutura com <main>, <section> e <article>

As tags <main>, <section> e <article> são elementos semânticos importantes para organizar o conteúdo em uma página HTML. A tag <main> envolve o conteúdo principal que é único na página, ajudando navegadores e leitores de tela a entenderem o foco principal. A tag <section> define seções temáticas do conteúdo, agrupando partes relacionadas, enquanto a tag <article> representa conteúdos independentes, como postagens de blog, notícias ou comentários, que podem ser reutilizados ou distribuídos separadamente.


Tags de Mídia

As tags de mídia em HTML permitem incorporar diferentes tipos de conteúdo multimídia diretamente na página, enriquecendo a experiência do usuário. A tag <img> é usada para exibir imagens, enquanto as tags <audio> e <video> possibilitam a inserção de áudio e vídeo, respectivamente. Estas últimas oferecem controles nativos para reprodução, pausa e volume, e suportam múltiplos formatos via elementos <source>. Caso o navegador não suporte esses elementos, é possível exibir uma mensagem alternativa.

A tag <iframe> permite incorporar conteúdo de outras páginas web, como vídeos do YouTube, mapas do Google Maps ou até documentos externos. Ela cria um quadro embutido na página e é muito usada para exibir recursos interativos hospedados em outros sites.

Outras Tags Sugeridas

Além das tags básicas, existem outras que ainda são úteis ou importantes para conhecer, mesmo que algumas estejam obsoletas. A tag <abbr> é usada para indicar siglas ou abreviações, exibindo uma descrição ao passar o mouse. A tag <font>, embora considerada obsoleta, ainda pode ser encontrada para alterar cores e estilos de texto, mas recomenda-se usar CSS para isso. Já a tag <del> indica que um texto foi removido ou alterado, sendo útil em documentos que passam por revisões.
